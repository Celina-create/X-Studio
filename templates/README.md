# Templates

Copy-paste assets. Each file is a ready-to-use template with placeholder fields marked `{{like_this}}`.

## Index

| Folder | What's inside |
|---|---|
| `cold-email/` | Subject + body templates for outreach, grouped by audience type |
| `landing-page/` | Landing page skeletons (markdown → HTML) for quick campaign pages |
| `issue-forms/` | `.yml` files for GitHub Issue Forms you can drop into any repo |

## Usage

These are starting points, not finished assets. Expect to rewrite at least 30% of any template to fit your specific context. The value is in the *shape*, not the exact words.

## Status

Folders are populated as case studies finish. If you need a template that isn't here yet, [open an issue](https://github.com/Celina-create/X-Studio/issues/new).
